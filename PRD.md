# Image Background Remover - MVP 需求文档

**版本：** v0.1  
**日期：** 2026-03-27  
**状态：** 草稿

---

## 1. 产品概述

一个在线图片背景去除工具，用户上传图片后自动去除背景，下载透明 PNG。

**核心价值：** 快、简单、免费用（限额内）

---

## 2. 目标用户

- 电商卖家（商品白底图）
- 设计师（快速抠图）
- 普通用户（证件照、头像处理）

---

## 3. 功能范围（MVP）

### 3.1 核心功能（P0，必须上线）

| 功能 | 描述 |
|------|------|
| 图片上传 | 支持点击上传 / 拖拽上传，格式：JPG、PNG、WEBP，大小 ≤ 10MB |
| 自动去背景 | 调用 Remove.bg API 处理，返回透明 PNG |
| 结果预览 | 处理完成后展示原图 vs 结果图对比 |
| 下载结果 | 一键下载透明 PNG |
| 错误提示 | 上传失败 / API 失败时给出明确提示 |

### 3.2 不在 MVP 范围（后续迭代）

- 替换背景色 / 背景图
- 批量处理
- 用户账号系统
- API 开放接口
- 历史记录

---

## 4. 技术方案

| 层级 | 技术选型 |
|------|----------|
| 前端 | Next.js 14 + Tailwind CSS |
| API 代理 | Next.js Edge API Route |
| 抠图服务 | Remove.bg API |
| 图片存储 | 无（内存中转，不落盘） |
| 部署 | Cloudflare Pages |

**数据流：**
```
用户上传图片 → Edge API Route → Remove.bg API → 返回 PNG → 用户下载
```

---

## 5. 页面设计

### 5.1 主页（唯一页面）

**布局结构（从上到下）：**

1. **Header**
   - Logo + 产品名
   - 简短 slogan（如：Remove image backgrounds instantly）

2. **上传区域**
   - 大尺寸拖拽区（虚线边框）
   - 支持点击触发文件选择
   - 提示文字：支持 JPG / PNG / WEBP，最大 10MB

3. **处理状态**
   - 上传中：进度提示
   - 处理中：loading 动画 + "Removing background..."
   - 完成：展示结果

4. **结果展示**
   - 左：原图缩略图
   - 右：去背景结果图（棋盘格背景表示透明）
   - 下载按钮（醒目，Primary 色）
   - "Try another image" 重置按钮

5. **Footer**
   - Powered by Remove.bg
   - 简单版权信息

### 5.2 交互状态

| 状态 | UI 表现 |
|------|---------|
| 初始 | 显示上传区域 |
| 上传中 | 上传区显示文件名 + loading |
| 处理中 | 全屏或区域 loading，禁止重复上传 |
| 成功 | 显示对比图 + 下载按钮 |
| 失败 | 红色错误提示，可重试 |

---

## 6. 接口设计

### POST /api/remove-bg

**Request：**
- Content-Type: multipart/form-data
- Body: `image`（File）

**Response（成功）：**
- Content-Type: image/png
- Body: 透明 PNG 二进制流

**Response（失败）：**
```json
{ "error": "错误描述" }
```

**错误码处理：**

| 场景 | 提示文案 |
|------|---------|
| 文件过大 | "图片不能超过 10MB" |
| 格式不支持 | "请上传 JPG / PNG / WEBP 格式" |
| API 额度耗尽 | "服务繁忙，请稍后再试" |
| 网络错误 | "处理失败，请重试" |

## 7. 非功能需求

| 指标 | 目标 |
|------|------|
| 处理时长 | ≤ 5 秒（取决于 Remove.bg 响应） |
| 移动端适配 | 响应式，支持手机上传 |
| 浏览器兼容 | Chrome / Safari / Firefox 最新版 |
| 安全 | API Key 仅在服务端，不暴露给前端 |
